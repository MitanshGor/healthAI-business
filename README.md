# HealthPulse AI - Business Website

## 🚀 Business Overview

**HealthPulse AI** is a revolutionary AI-powered fitness platform that transforms raw health data into personalized, actionable workout plans. We're solving the critical problem of data overload in the fitness industry by providing intelligent, personalized guidance that actually works.

### 🎯 The Problem
Ever feel lost looking at your health data, unsure how it translates into the right workout for you? Millions of people with health devices struggle to understand their data and get actionable guidance, leading to:
- **85% app abandonment** within 3 months
- **73% confusion** about health data meaning
- **Only 23%** stick to their fitness goals
- **40% lower engagement** with generic advice

### 💡 Our Solution
HealthPulse AI turns your data, schedule, and goals into a personalized workout plan — so you know exactly what to do today to feel stronger and healthier.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or pnpm
- Git

### Installation
```bash
# Clone the repository
git clone https://github.com/MitanshGor/healthAI-business.git

# Navigate to project directory
cd health-pulse-ai-business-website

# Install dependencies
npm install
# or
pnpm install

# Run development server
npm run dev
# or
pnpm dev
```

### Build for Production
```bash
# Build the project
npm run build

# Start production server
npm start
```

## 🏗️ Architecture

### Frontend Architecture
```
components/
├── common/          # Reusable UI components
├── features/        # Feature-specific components
├── layout/          # Layout and navigation components
├── ui/              # Base UI components
└── providers/       # Context providers

app/
├── api/             # API routes
├── dashboard/       # Dashboard pages
└── globals.css      # Global styles
```

### Tech Stack
- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Deployment**: Vercel
- **Database**: Supabase (PostgreSQL)
